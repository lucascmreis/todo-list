# todo-list
Pequena aplicação de marcação de tarefas, ou famoso TODO list.  Para treinar um pouco mais sobre manipulação do estado no React


## O que faz esse app ?

- Adicionar uma nova tarefa
- Remover uma tarefa
- Marcar e desmarcar uma tarefa como concluída
